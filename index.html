How it works

One protocol. Three steps. No trust required.

The same cryptographic pipeline every time. No shortcuts, no exceptions.

01
Seal
Content gets hashed with SHA256, signed with ECDSA, and anchored to the Constellation Network blockchain. From this moment forward, any alteration is mathematically detectable — by anyone.
→
02
Bundle
The content, signature, PDF chain of evidence report, and full audit trail get packaged into one sealed file with a signed SHA256 sidecar. Everything needed to verify is inside. No external system required.
→
03
Verify
Drop the bundle here or open verify.html inside the zip. No account. No software. No us. The proof is self-contained and permanent. It was true before we existed and stays true if we disappear.
